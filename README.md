# Chrome Portable (Policy Bypass Fork)

このリポジトリは、組織のグループポリシー (GPO) による制限を回避して起動することを目的とした、ポータブルな Google Chrome の独立フォークです。

- **[Chrome-portable-v1.1.4.zip (GitHub Releases)](https://github.com/tenma2066-tech/Chrome-portable/releases/download/v1.1.4/Chrome-portable-v1.1.4.zip)**

---

## ⚠️ 重要事項（必ずお読みください）
- **更新禁止**: Chrome 本体の更新（設定画面からのアップデート等）は**絶対に行わないでください**。バイパス機能が壊れる可能性があります。
- **独立性**: PC 本体の設定（レジストリ）を書き換えても、終了時や復元用スクリプトで元に戻せる設計になっており、学校や会社の PC を汚さずに使えます。
- **初回準備**: ダウンロードした ZIP ファイルは、必ず**「すべて展開（解凍）」**してから使用してください。

---

## 🚀 起動方法（使いかた）

パソコンの環境（管理者権限があるかどうか）に合わせて、**どちらか片方**を実行してください。

### 【方法 A】 管理者権限（パスワード）が使える場合
最も確実で強力な制限回避方法です。

1.  フォルダ内にある **`Run.bat`** をダブルクリックして実行します。
2.  「ユーザーアカウント制御」の画面が出たら **「はい」** を押してください。
3.  黒い画面（コンソール）が出て、自動的に制限の解除処理が始まります。
4.  数秒待つと、自動的に Google Chrome が起動します。

### 【方法 B】 管理者権限が使えない場合
学校の PC など、管理者パスワードが分からない場合でも使える方法です。追加のソフト（Python 等）は不要です。

1.  フォルダ内にある **`patch.bat`** をダブルクリックして実行します。
2.  黒い画面が出ます。初回は `chrome.dll` という大きなファイルを書き換えるため、**数秒〜数十秒**かかります。
3.  画面に `[+] Patch applied successfully`（パッチ適用完了）と表示されるまで待ってください。
4.  完了すると、自動的に Google Chrome が起動します。
    *   ※2回目以降は、既に適用済みであればすぐに起動します。

## ファイルの役割
- `Run.bat`: 管理者向けのメインランチャー。レジストリを保護して起動します。
- `patch.bat`: 管理者権限がない人向けのランチャー。DLL ファイルを書き換えて起動します。
- `GoogleChromePortable.exe`: Chrome をポータブルとして動かすための本体です。
- `App/`: Chrome のプログラム本体が入っています。
- `Data/`: あなたの履歴、ブックマーク、パスワードなどが保存されるフォルダです。
- `README.md`: この説明書です。
